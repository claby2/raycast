# raycast

2D ray casting with SDL2.

## Getting Started

### Dependencies

+ [SDL2](http://libsdl.org/download-2.0.php).
+ CMake

### Usage

```console
git clone https://github.com/claby2/raycast.git
cd raycast
cmake .
make
raycast
```

## Testing

Testing uses `assert.h` which is part of the standard library.

```console
git clone https://github.com/claby2/raycast.git
cd raycast
cmake .
make raycast_test
raycast_test
```
